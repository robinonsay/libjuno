= Map Module

== Purpose

The Map module (`map_api.h`) provides an open-addressed hash map built on the Array and Pointer traits.
It uses linear probing over a caller-supplied array and is parameterized by hash, equality, and null-detection callbacks.

== Data Structures

=== Map Root (`JUNO_MAP_ROOT_T`)

[source,c]
----
struct JUNO_MAP_ROOT_TAG JUNO_MODULE_ROOT(JUNO_MAP_API_T,
    const JUNO_MAP_HASHABLE_POINTER_API_T *ptHashablePointerApi;  // Hash + null-check
    const JUNO_VALUE_POINTER_API_T *ptValuePointerApi;             // Key equality
    JUNO_DS_ARRAY_ROOT_T *ptHashMap;                               // Backing array
);
----

=== Hashable Pointer API (`JUNO_MAP_HASHABLE_POINTER_API_T`)

[cols="1,3"]
|===
| Function | Description

| `Hash(tItem)` | Compute a `size_t` hash for the key.
| `IsValueNull(tItem)` | Return `true` if the slot is logically empty.
|===

=== Map API (`JUNO_MAP_API_T`)

[cols="1,3"]
|===
| Function | Description

| `Get(ptJunoMap, tReturnItem)` | Look up by key; return pointer to matching entry.
| `Set(ptJunoMap, tItem)` | Insert or overwrite an entry.
| `Remove(ptJunoMap, tKey)` | Remove an entry by key.
|===

== Interface Design

`JunoDs_MapInit(ptMapRoot, ptHashablePointerApi, ptValuePointerApi, ptArray, pfcnFailureHandler, pvUserData)`::
Wires the map vtable, stores all three dependency APIs and the backing array, then validates via `JunoDs_MapVerify`.

`JunoDs_MapGet(ptJunoMap, tItem)`:: Linear-probe lookup by key.
`JunoDs_MapSet(ptJunoMap, tItem)`:: Linear-probe insert or overwrite.
`JunoDs_MapRemove(ptJunoMap, tKey)`:: Linear-probe removal.

`JunoDs_MapVerify(ptMap)`::
Inline verification: checks all API pointers (Hash, IsValueNull, Equals, Set, Get, Remove), backing array, and delegates to `JunoDs_ArrayVerify`.

== Algorithm Descriptions

=== Set (Insert/Update)

. Compute `index = Hash(key) % capacity`.
. Probe linearly: if slot is empty (`IsValueNull` returns true), copy entry via `SetAt`.
. If slot contains a matching key (`Equals` returns true), overwrite via `SetAt`.
. Continue probing with wrap-around until an empty slot or match is found.
. Return `JUNO_STATUS_TABLE_FULL_ERROR` if the probe traverses the full table.

=== Get (Lookup)

. Compute `index = Hash(key) % capacity`.
. Probe linearly: if slot matches key (`Equals`), return pointer descriptor via `GetAt`.
. If slot is empty (`IsValueNull`), key does not exist — return `JUNO_STATUS_DNE_ERROR`.
. If full traversal without match, return `JUNO_STATUS_TABLE_FULL_ERROR`.

=== Remove

. Compute `index = Hash(key) % capacity`.
. Probe linearly: if slot matches key, reset via `RemoveAt` and return success.
. If empty slot encountered before match, key was not present — return success (no-op).

All operations are O(1) average, O(n) worst-case under high load or clustered collisions.

== Error Handling

* `Get` returns `JUNO_STATUS_DNE_ERROR` when key not found.
* `Set` returns `JUNO_STATUS_TABLE_FULL_ERROR` when no empty slot available.
* Pointer type mismatches return `JUNO_STATUS_INVALID_TYPE_ERROR`.
* All operations propagate errors from hashing, equality checks, and array access.
* All operations call `Verify` at entry.

== Design Rationale

Open addressing with linear probing was chosen for cache-friendliness and zero-allocation operation over a single contiguous array.
The map is parameterized by three callback APIs (hash, equality, null-check) rather than hard-coding a specific key type, keeping it type-agnostic and reusable.
